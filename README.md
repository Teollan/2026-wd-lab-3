# Розробка Web-додатка засобами Javascript/VueJS

**Студент:** Сергій Козлов

**Група:** КВ-33

## Мета

Ознайомитись із засобами фреймворка VueJS та навчитись створювати асинхронні запити до Web-сервера.

## Загальне завдання

Адаптувати програмний код ЛР№2 до вимог фреймворка VueJS та забезпечити завантаження необхідних даних з Web-сервера.

## Тематика додатку

Блог з коментарями (**BlogSpace**).

## Опис додатка

Web-додаток для ведення блогу: реєстрація та автентифікація користувачів, перегляд стрічки повідомлень, створення, редагування та вилучення власних повідомлень, а також додавання коментарів до повідомлень. Дані зберігаються на сервері у базі даних; клієнт взаємодіє з сервером через асинхронні запити.

## Інструменти розробки

- **Клієнт:** Javascript, VueJS 3 (Composition API), Vue Router, Pinia, Axios, Tailwind CSS, Vite
- **Сервер:** NodeJS, Fastify, Prisma ORM, PostgreSQL, JWT-автентифікація
- **Інфраструктура:** Docker (PostgreSQL), TypeScript

> Примітка: завдання дозволяє обрати інший серверний фреймворк замість Express — використано Fastify.

## Структура репозиторію

- `frontend/` — клієнтський додаток на VueJS
- `backend/` — серверний застосунок (Fastify + Prisma)
- `docs/` — текст завдання лабораторної роботи
